# 楔子

等午餐外卖的间隙，闲逛 x，偶遇 Confirmo by @yetong。搜 character 的时候，发现了原神。想象自己玩了5，6年原神，也算是个资深玩家，竟然没有用过原神的主题。于是在 vscode themes 里一番搜索，发现主题还真不少。挑了一个 star 最多的，准备安装，突然又想知道，自己用过多少个主题呢？现在用的又是哪个主题呢？filter 下已安装的主题，然后突然就想找个地方记录下。那，既然要记录，干脆多记录一点吧。
看吧，人类的思维就是这么跳脱，这是 AI 远远比不了的。

# 我的 2025 年

今年过的很丰富多彩。

用 swift ui 开发了几个小工具：

1. 生成 .npmrc 文件（mac）
2. 管理余额（iphone）：最开始的 idea 是记录兴趣班课程余额，后来觉得，既然做了，就做成一个通用的余额管理工具吧。嗯嗯，现在也可以记录理发店次数卡余额。

自学了维修知识。购买工具画了不少钱，但是相比请师傅上门的话费和不确定性来说，我觉得很超值。

1. 更换了厨房水槽。学会了水槽的安装和拆卸，水管的连接和密封。
2. 更换了几处柜门铰链，学会了铰链的安装和调整，分类。后续又订购了几块门板更换掉了腐烂的门板。
3. 更换了洗衣机密封圈，忙了一个下午，胳膊酸了好几天。不得不说西门子的洗衣机真的重。
4. 给自己的衣柜装了两块隔板，没有买伸缩隔板，直接买了模板和层板托。
5. 用角钢做了一个洗手台架子，又买了海洋般，自制打磨，上木蜡油。

感觉现在完全可以接一些简单的维修活儿了，哈哈哈。

自驾露营若干次：

1. 去了九龙山，在山上吃了凉面，嘎嘎香。
2. 去了农场，孩子体验了割稻子，捉鱼。提前买了移动电源，在露营地吃了一次热腾腾的泡面，香。
3. 开车去了嘉兴，参加了公司的亲子游，自驾的体验非常好，不用等人等车，不用着急赶时间。
4. 开车去了很多以前没去过觉得不好停车的地方。

说的不是dev吗？怎么会提到维修和露营。我把生活也当作dev，每一次开拓新领域，学习新技能，都是在提升自己解决问题的能力，不是吗？

# 用过的工具：

## vscode

1. 主题： xcode
2. copilot chat，要研究好用的 instruction 哦。

## lightsail + godaddy

自助梯子，搭建方法参考耗子哥的博客。想睁眼看世界的人都要学会爬梯子。

## 维修工具

1. 电钻：某音评价的垃圾 得力 手电钻，一百不到，够用就好，拧螺丝，自攻螺丝，都能干。
2. 电钉枪：129元的“垃圾”，非常实用，我用来在墙上钉一些不太重的东西。
3. 冲击钻：绿林，160元，能在瓷砖，水泥墙打孔，够用。
4. 万用宝：威克士，500多，有点贵，实用，用来开孔，切割，打磨。
5. 其他的，线钳，活动扳手。
